# flipkart-sentiment-analysis
Sentiment analysis on Flipkart product reviews using ML
This project performs sentiment analysis on flipkart product review using machine learning
## Objective
To classify customer reviews into positive, Nehative, or Neutral based on ratings and text
## Technologies used
Python
Pandas
Scikit-learn
TF-IDF  Vectorizer 
Multinomial Naive Bayes
## Steps involvedC
Data loading
Data Preprocessing
Sentiment from Ratings
Train-Test Split
Feature Extraction (TF-IDF)
Model Training
Model Evaluation (Accuracy & Confusion Matrix)
## Result
 - Accuracy: **82.7%**
## Sample Prediction
Input:
"This product is very good"
Output:
Positive
